Detailed Description

Class for a 1, 2 or 3-dimensional universe of Particles. Supports simulation according to Str√∂mer-Verlet's algorithm for electrostatic forces and output to .vtk file for visualisation.

The particles are stored inside deques into a grid (1, 2 or 3 D) this grid is strored in a 1D vector. To factorise code as much as possible we had to store the x axis into the last dimension of the grid so that for example grid[y * grid_size[1] + x] gets the deque (x, y) in 2D and the deque x in 1D when grid_size[1] is 0

Simulates the evolution of the universe.

At every step dt of time, updates positions of particles according to St√∂rmer-Verlet's algorithm.
i.e. calls update_forces then update_positions and adds dt to t until t reaches t_end.
Also writes universe's state to a .vtk file.
